Web Directories Tab 

The web directories tab allows you to perform directory enumeration against a web server.

How to use

  • Type the target URI e.g. http://www.mydomain.com/test
    • You can select http, as well as https 
  • Type the port number e.g. port 80, 443 or any other port
  • Specify the directory names a default list of 58658 is loaded
  • Select "Start"

What to expect

While the process is running, you can sort the results by the corresponding header column.

  • To sort by response code (e.g. 200, 404, 302), click on the Code row inside the Output table

Advanced options

While enumerating directories, an error might occur. This is often due to a server failing to respond for a short period of time, TCP collisions, etc.

You can set system preferences to stop or continue upon the occurence of an error. This option is located in:

  • Preferences
    • Fuzzing
      • Directories: While fuzzing if an error occurs, Continue 
       

Directory Enumeration:

The process of identifying existing directories on a web server.

 

This is achieved by recursively requesting directory names from a given list.

 

Typical findings include the discovery of an administrator interface (e.g. /admin) as well as directories that are not linked to the web site.
