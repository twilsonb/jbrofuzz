Open Source Tab 

This tab allows you to harvest email addresses that are in the public domain, given the Fully Qualified Domain Name (FQDN) that is under scrutiny.

This is achieved by submitting a total of five (5) requests to Google Web Search and Google Groups with the FQDN provided and filtering through the responses.

How to use

Valid formats are: “mydomain.com”, “myuni.ac.uk”, etc. Do not include the ‘@’ symbol or any of the quotes " in the domain.

  • You will require an Internet Connection
  • Type in the domain in question (e.g. mydomain.com) 

What to expect

A list of all successfully recovered email addresses from the public domain will be displayed in the Output Panel.

Advanced options

E-mail addresses returned are not limited to the domain specified, but also include addresses which have been indexed along side the domain in question.

      

Open Sourcing:

Describes the process of recovering information about a given organisation, website or individual that is available in the public domain.

This is particularly useful during fuzzing at it can provide:

  • Username Information
  • Email / Username format
