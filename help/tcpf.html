Fuzzing Tab

This tab allows you to perform fuzzing, by specifying a URL and clicking "Start". It represents the main tab of JBroFuzz.

A number of payloads can be added to the message request being submitted:

- Highlight any part of the "Request" panel

- Click " + " to add a fuzzer and select "Start"

JBroFuzz will iterate (either recursively or replacively) through the payloads of the fuzzer selected, producing the results in the "Output" panel at the bottom.

Details for particular payloads and corresponding fuzzers can be found in the "Payloads" tab.

HelloGoogle Tutorial

Similarly to HelloWorld examples used in programming, this section demonstrates a HelloGoogle, i.e. how to send a single request to a google web server using (in this case) JBroFuzz.

  • In the URL Panel (top left section) replace "http://localhost" with " http://www.google.com "
  • From the Toolbar menu select "Start"< icon)>

That's it! What happened?

A connection to www.google.com has been made, supplying as a request, the text in the request field.

  • Double-click on the line added to the output panel to examine the request and response received

  • Repeat the same exercise, supplying as URL: https://www.gmail.com

Fuzzing Workflow Process by Example

We have examined how a single request is sent to a web server over a standard port. In order to perform a fuzzing operation, a generator needs to be added to part of the request submitted. JBroFuzz will then iterate through all the possible requests, submitting them sequentially to web server selected.

Exercise: "Examine which HTTP major protocol versions the URL https://www.cia.gov supports

  • In the URL Panel (top left section) type https://www.cia.gov
  • In the Request Panel (top left section) highlight using the mouse the number '1' in the first line: "GET / HTTP/1.0"
  • In the Toolbar, select the "Add Generator" button (labelled with a cross icon) and double click on "NUM-OCT"
  • In the Toolbar, select "Fuzz" (labelled with a play icon)

What happened?

We selected to add the number octal generator to the digit '1' that is part of the first line of the request we are submitted to https://www.cia.gov .

Thus a total of 8 requests will be submitted to the web server, consecutively:

  • "GET / HTTP/0.0"
  • "GET / HTTP/1.0"
  • ....
  • "GET / HTTP/7.0"

Notice that we get a response from cia.gov of size 31882 for the first request, being "GET / HTTP/0.0" !!!

General How to Use

In order to perform fuzzing, you will need to add one or more generators to the request being submitted:/font>

  • Select (highlight) part of the message request
  • Select "Add a Generator"
  • Select the type (e.g. DEC for "Decimal Fuzz Type")

As an example, in the case you selected the above and highlighted a single character from the request, a total of 10 requests iterating through digits 0 to 9 will be sent to the target.
