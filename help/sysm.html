System Tab 

The system tab provides information about any errors, or problems that might have occured, during startup, runtime and/or previous shutdown of the application.

Generally, information logged in the system tab relates to recoverable errors, as well as general issues.

What to expect

The system tab is updated automatically, during the occurence of an action that has been worthy of displaying.

  • Each line begins with the date and time that the action has been logged 
  • This follows the tab under which the action relates to
  • Finally the action logged is displayed

Advanced options

By selecting the info button, you can see a breakdown of information relating to:

  • The Java Environment
  • The User running JBroFuzz
  • The Operating System 
       

Logging & Saving:

The information displayed within the system tab is not written to file.

 

If you require this information to be saved, you have to perform a copy/paste.
