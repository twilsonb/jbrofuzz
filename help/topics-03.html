Headers Window  

The Headers Window provides a number of browser headers collected from different browsers on a variety of different operating systems.

 

Getting Started: Click on a particular node, on the left hand side, to see its header and corresponding properties.

This tab is provided, as many web applications respond differently to different browser impersonation attacks.

 

Note: As of version 1.9 of JBroFuzz, the old, Headers Tab is actually a separate window.

You can access the Headers Window, by Options -> Browser Headers, or selecting [Ctrl + H]
