JBroFuzz Keyboard Shortcuts

Tab Shortcuts:

 

Fuzzing: 

Ctrl + Enter Start Fuzzing
Ctrl + Backspace Stop Fuzzing
Ctrl + Equals Add a Fuzzer
Ctrl + Minus Remove a Fuzzer
Ctrl + Q Clear all Output
Ctrl + L Open Location
Ctrl + O Open .jbrofuzz File
Ctrl + S Save as .jbrofuzz File

 

Graphing: 

Ctrl + Enter Find all Directories

 

System: 

Ctrl + Enter Perform a System HealthCheck
Ctrl + Equals Add a Timestamp to the System Log

Global Shortcuts:

Ctrl + H Browser Headers Window
Ctrl + E Encode/Hash Window
Ctrl + P Preferences
Ctrl + 0 (Zero) About...
Ctrl + 1 (One) Exit.
