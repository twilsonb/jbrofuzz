JBroFuzz FAQ - What is a Fuzzer?

Q: What is a Fuzzer?

 

A fuzzer contains a particular character set which will be tested against the specified range of the request.

 

Some examples of generators:

Sending all requests of : 0000 to 1111 (binary generator over 4 characters)

Sending all requests of : 00 to 99 (decimal generator over 2 characters).

To get a better understanding of how fuzzers work, view the tutorial demo that is available for download.

 

Q: Where are all the fuzzer definitions stored?>

 

All fuzzer definitions are loaded at start time, from the fileneed to select the part of the request that you would like to fuzz.

This then gives you the ability to add a number of generators on the specified range.

A s a result, the generator gets added to the generators list on the right hand side.
